Alhaiz Breizh

Alhaiz Breizh est un projet fictif réalisé en deuxième année de BUT Informatique. Ce site offre aux utilisateurs la possibilité de rechercher des hébergements répondant à leurs critères, comme la localisation, le type de logement ou le budget. L'objectif principal du projet était de mettre en pratique des compétences acquises et de concevoir un outil adapté aux besoins des utilisateurs.

Technologies utilisés : HTML / CSS / PHP / JavaScript

Le cheval blanc

Ce projet, réalisé en première année de BUT Informatique, consistait à développer une application répondant aux besoins d’un client pour la gestion de réservation d’hôtel. L’objectif était de concevoir une solution fonctionnelle en appliquant des méthodologies adaptées, de la modélisation à la réalisation. Ce travail a permis de structurer les données, de créer une interface intuitive, et de mettre en pratique les compétences acquises en équipe.

Technologies utilisés : UML / Java / JavaFX
