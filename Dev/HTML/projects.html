Alhaiz Breizh

Alhaiz Breizh est un projet fictif réalisé en deuxième année de BUT Informatique. Ce site offre aux utilisateurs la possibilité de rechercher des hébergements répondant à leurs critères, comme la localisation, le type de logement ou le budget. L'objectif principal du projet était de mettre en pratique des compétences acquises et de concevoir un outil adapté aux besoins des utilisateurs.

Technologies utilisés : HTML / CSS / PHP / JavaScript

Pokédex Pokémon

Ce projet, réalisé en deuxième année de BUT Informatique, consistait à développer un Pokedex Pokémon qui est une application web pour répertorier et afficher des informations détaillées sur les Pokémon. Elle permet aux utilisateurs de consulter des données telles que les types, les statistiques et les évolutions des Pokémon.

Technologies utilisés :JavaScript / HTML / CSS
